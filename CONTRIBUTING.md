# SIGINT Hub Katkı Sağlama Rehberi

Bu projeye katkıda bulunmak istediğiniz için teşekkürler! Modern Sinyal İstihbaratı dünyasını daha iyi anlamak ve geliştirmek için aşağıdaki adımları takip edebilirsiniz.

## Nasıl Katkı Sağlayabilirim?

1.  **Hata Bildirimi:** Bir hata (bug) bulursanız lütfen "Issue" açarak bildirin.
2.  **Özellik Önerisi:** Yeni simülasyonlar (PSK dekodlama, LoRa analizi vb.) için önerilerde bulunun.
3.  **Kod Geliştirme:**
    *   Depoyu (repo) fork'layın.
    *   Yeni bir dal (branch) oluşturun (`git checkout -b feature/yeni-ozellik`).
    *   Değişikliklerinizi yapın ve test edin.
    *   Pull Request (PR) gönderin.

## Kod Standartları

*   Kodlar anlaşılır ve yorum satırlarıyla desteklenmiş olmalıdır.
*   Özellikle DSP (Dijital Sinyal İşleme) bloklarında matematiksel açıklamalar eklenmelidir.
*   README estetiğini bozmayacak şekilde görsel dökümantasyon tercih edilir.

## Etik Kurallar

Bu proje sadece eğitim amaçlıdır. Katkılarınızın bu çerçevede kaldığından ve yasal sınırları ihlal etmediğinden emin olun.
